## 1.4 - O que é uma interrupção?

Primeiro quero que você reflita, como o meu computador sabe na minha linguagem de alto nível, por exemplo, o python que esse código aqui:

```python
print("Hello World!")
```

Vai produzir essa saída:

```shell
Hello World!
```

Ele não sabe, computadores não pensam e não tem conciência, e aqui vai uma das primeiras lições que você deve aprender sobre computadores, eles só fazem o que mandamos.
Ok, ok, então como ele sabe que eu estou mandando ele imprimir isso na minha tela?
Agora sim, boas perguntas começaram a aparecer! O python, sofre um processo de <a href="https://pt.wikipedia.org/wiki/Linguagem_interpretada" target="_blank">interpretação</a>,
onde, um outro programa escrito em uma linguagem <a href='https://pt.wikipedia.org/wiki/Linguagem_compilada' target="_blank">compilada</a>,
chamado interpretador vai interpreta-lo. O ponto é, mesmo o python precisa dizer para o sistema o que ele quer executar, do que adianta ele fazer todo trabalho sujo e não ter para quem mostrar?
